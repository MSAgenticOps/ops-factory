# CWD-1558 将同一变量用于多种用途

**描述**
一个变量应该只用于一种用途，否则，将同一变量用于多种用途时，不但会降低代码的可读性，而且可能隐藏其他问题。

**语言: **CPP,JAVA,PYTHON,ARKTS

**严重等级**
一般

**cleancode特征**
安全,可靠

**业界缺陷**
