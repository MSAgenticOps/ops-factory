# CWD-1466 函数的参数过多

**描述**
函数或方法需要太多的参数来执行其功能，意味着其功能不单一，难以理解和维护。此外，由于参数太多，调用该函数或方法时也会变得更加困难，这可能会导致错误的发生。因此，应该限制参数个数。例如：C语言中函数的参数个数。建议不超过5个。

**语言: **C,CPP,JAVA,PYTHON,GO,JAVASCRIPT,TYPESCRIPT,ARKTS,LUA,PHP
