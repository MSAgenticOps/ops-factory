# CWD-1400 混合编码处理不当

**描述**
当同一输入使用多个不同的编码方式时，但没有正确地处理它们之间的转换关系，导致出现了意外的结果。常见的编码方式包括：URL编码、HTML编码、Unicode编码等。

**语言: **CPP,JAVA,SHELL
