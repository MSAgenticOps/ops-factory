# CWD-1375 父类有虚析构，子类没有虚析构

**描述**
父类有虚析构方法，但是子类没有虚析构方法，会导致子类的资源无法释放，进而导致内存泄漏。

**语言: **CPP,JAVA

**严重等级**
提示

**cleancode特征**
可靠

**业界缺陷**
