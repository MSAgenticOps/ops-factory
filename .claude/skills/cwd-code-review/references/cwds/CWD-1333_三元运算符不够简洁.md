# CWD-1333 三元运算符不够简洁

**描述**
使用逻辑复杂的三元运算符表达式，让代码变得令人困惑。例如：condition ? true : expr 可简化为 condition || exprcondition ? false : expr 可简化为 !condition && exprcondition ? expr : true 可简化为 !condition || exprcondition ? expr : false 可简化为 condition && expr"
