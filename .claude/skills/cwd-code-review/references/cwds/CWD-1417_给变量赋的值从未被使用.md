# CWD-1417 给变量赋的值从未被使用

**描述**
在不使用的情况下对变量进行赋值，代码给变量 `variable` 赋值，然后在不使用它的情况下又覆盖该变量的值。

**语言: **C,CPP,JAVA,ARKTS,MICROCODE

**严重等级**
提示

**cleancode特征**
可靠
