# CWD-1415 空代码块

**描述**
源代码包含一个不包含任何代码的块。空代码块可能出现在条件、函数或方法定义、异常处理程序等的主体中，包括空异常块、空同步块。虽然空代码块可能是有意的，但也可能表示实现不完整、意外删除代码、意外的宏扩展等。对于某些编程语言和构造，语法可能允许空代码块。但块中缺少任何行为可能会违反约定或API，从而导致错误。
