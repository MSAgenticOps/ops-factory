# CWD-1314 过于依赖全局变量

**描述**
全局变量是在所有作用域都可访问的变量。过度依赖全局变量会带来一系列风险，如紧耦合、调试和维护困难，因此，应谨慎使用全局变量，尽量不用或少用全局变量。

**严重等级**
