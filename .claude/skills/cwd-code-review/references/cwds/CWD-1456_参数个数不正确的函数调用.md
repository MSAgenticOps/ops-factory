# CWD-1456 参数个数不正确的函数调用

**描述**
调用的函数的参数个数过多或过少，会产生不正确的行为。例如：当传递给格式化字符串函数的参数个数与格式字符串中的格式说明符个数不匹配时，也会导致未定义的行为。int x = 10; printf("%d %d", x);  // 错误：两个格式说明符，但只有一个参数"
