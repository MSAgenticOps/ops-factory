# CWD-1564 交付代码中包含TODO/TBD/FIXME等注释

**描述**
代码包含的注释表明存在bug、不完整的功能或弱点。如：BUG, HACK, FIXME, LATER, LATER2, TODO。

**语言: **C,CPP,JAVASCRIPT,TYPESCRIPT,ARKTS,PHP

**严重等级**
一般

**cleancode特征**
安全,可靠

**业界缺陷**
