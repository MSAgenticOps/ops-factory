# CWD-1422 布尔值冗余检查

**描述**
使用条件语句判断一个布尔变量并返回一个布尔值，而这个布尔值已经是由条件表达式本身决定的，增加了不必要的复杂性。例如：


- `if (expression) { return true; } else { return false; }`  可以简化为：`return expression;`- `return (b ? true : false);` 可以简化为：`return b;`简化后代码更简洁、更易读懂。
**语言: **PHP

提示
