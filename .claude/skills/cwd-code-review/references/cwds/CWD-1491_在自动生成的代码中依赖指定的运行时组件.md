# CWD-1491 在自动生成的代码中依赖指定的运行时组件

**描述**
自动生成的代码可能会受到运行时组件的安全性的影响。又因为是自动生成的代码，可能会使发现/修复问题变得更加困难。

**语言: **C,CPP,JAVA

**严重等级**
提示

**cleancode特征**
可靠
