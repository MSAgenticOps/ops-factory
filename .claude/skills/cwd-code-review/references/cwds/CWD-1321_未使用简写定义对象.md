# CWD-1321 未使用简写定义对象

**描述**
Vue在定义对象时，推荐使用简写语法。这种语法可以更简洁地定义对象，并且可以更好地表达对象的结构。例如，如果我们要定义一个包含属性`name`和`age`的对象，我们可以这样写：

```javascript
// 使用简写语法
const person = { name, age };

// 不使用简写语法
const person = { name: name, age: age };
```

可以看到，使用简写语法可以减少冗余代码，使代码更加简洁易读。
