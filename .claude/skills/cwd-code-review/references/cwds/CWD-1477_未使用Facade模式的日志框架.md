# CWD-1477 未使用Facade模式的日志框架

**描述**
专用日志工具与控制台打印相比，提供了更丰富的日志记录功能，且使用更加简单。日志打印推荐使用Facade模式的日志框架，如第三方slf4j、产品自研日志框架等，不要进行控制台打印（如：Java语言的System.out、System.err、C语言的printf、shell语言的echo 等）。
