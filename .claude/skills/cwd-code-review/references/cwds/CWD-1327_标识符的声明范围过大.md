# CWD-1327 标识符的声明范围过大

**描述**
如果标识符声明的范围比需要的范围大，则会降低代码的可读性，并可能被意外引用而产生预期之外的错误。因此，标识符在满足功能的同时，应在最小作用域内声明。

**语言: **C,CPP,JAVA

**严重等级**
提示

**cleancode特征**
可靠
