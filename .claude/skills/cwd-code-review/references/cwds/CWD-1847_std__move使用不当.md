# CWD-1847 std::move使用不当

**描述**
在 C++ 中，`std::move` 是一个强大的工具，但如果使用不当会导致各种问题。例如：对局部变量使用 `std::move`  破坏了类型一致性，阻止了编译器的返回值优化。正确的做法是直接返回局部变量。
