# CWD-1485 使用已过时或弃用方法

**描述**
代码中使用已过时或弃用（Obsolete）函数、tag属性等，这些可能不再被维护、修复或支持，可能会导致软件出现安全漏洞、崩溃或其他问题。因此，建议定期对代码进行审查，检测和修复使用过时函数的代码，并使用替代函数或最新的API，以确保代码的安全性和稳定性。

**语言: **JAVASCRIPT,TYPESCRIPT,ARKTS,LUA,PHP

**严重等级**
提示

**cleancode特征**
