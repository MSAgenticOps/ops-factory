# CWD-1414 无关代码

**描述**
产品包含的代码对执行不是必需的，即不进行状态更改，也没有改变数据或控制流的副作用，因此删除代码不会对功能或正确性产生任何影响。不相关的代码可能包括死代码、空代码块、由于优化而可能完全删除的代码等。

**严重等级**
提示

**cleancode特征**
可靠
