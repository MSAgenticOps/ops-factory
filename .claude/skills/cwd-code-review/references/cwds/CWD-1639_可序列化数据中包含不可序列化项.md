# CWD-1639 可序列化数据中包含不可序列化项

**描述**
当一个可序列化的对象包含非可序列化的对象时，可能会导致数据丢失、应用程序崩溃或安全漏洞。

**语言: **C,CPP,JAVA

**严重等级**
提示

**cleancode特征**
可靠
