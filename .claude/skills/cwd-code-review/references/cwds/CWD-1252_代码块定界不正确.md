# CWD-1252 代码块定界不正确

**描述**
代码未显式分隔包含2个或以上语句的代码块，从而产生逻辑错误。可能导致明显不正确的行为。

**语言: **C,CPP

**严重等级**
提示

**cleancode特征**
可读

**示例**
**案例1: 条件语句未使用大括号**
**语言: **C

**描述**
条件语句中有多条语句时，未使用大括号对这些语句进行限定。

**反例**
```c
if (condition==true)
    Do_X();
    Do_Y();
```

**正例**
```c
if (condition==true)
{
    Do_X();
    Do_Y();
}
```

**修复建议**
条件语句后必须使用大括号。
