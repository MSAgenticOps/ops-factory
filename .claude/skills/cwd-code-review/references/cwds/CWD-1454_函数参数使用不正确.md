# CWD-1454 函数参数使用不正确

**描述**
调用函数时，使用了错误的参数，或未按函数使用手册/函数使用约定来指定函数参数。包括：错误的参数类型、错误的参数个数，错误的参数顺序。

**语言: **CPP,JAVA

**严重等级**
提示

**cleancode特征**
可靠
