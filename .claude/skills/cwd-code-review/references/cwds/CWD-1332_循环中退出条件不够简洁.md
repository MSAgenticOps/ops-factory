# CWD-1332 循环中退出条件不够简洁

**描述**
使用复杂的循环退出条件，让代码变得令人困惑。例如：for (int i = 0 ; i < 10 ; i++) {if (i * i <= 25) {continue;}break; // 不符合：在循环结尾处break，可读性差}for (int i = 0 ; i < 10 ; i++) {if (i * i > 25) {break; // 符合：逻辑正确，方法意图比较清晰}}

**语言: **CPP,JAVA,ARKTS

**严重等级**
提示

**cleancode特征**
可靠
