# CWD 代码审查技能

本技能基于 **CWD（代码缺陷字典）** 提供代码审查能力。

## 概述

CWD 代码审查技能分析了源代码，以识别基于 CWD 字典中 200+ 分类模式的潜在缺陷、漏洞和代码质量问题。

## 目录结构

```
cwd-code-review/
├── SKILL.md                   # 主要技能定义
├── README.md                  # 使用说明
├── references/
│   ├── cwd_index.md           # 所有 CWD 类别索引
│   └── cwds/
│       ├── CWD-1002_内存分配大小未受限   # 某一CWD缺陷
└── templates/
    └── report_template.md     # 报告输出模板
```

## 功能特性

### 按需引用加载

引用按类别组织，按需加载：

```markdown
加载 CWD-1002 引用          # 单个类别
加载 CWD 内存安全类别        # 类别组
加载 CWD 安全引用            # 所有安全类别
```

### 支持的语言

- **C/C++**：全面支持内存、并发、安全类别
- **Java**：全类别综合支持
- **Python**：安全、代码质量、输入验证
- **JavaScript/TypeScript**：Web 安全、代码质量
- **Go**：内存安全、并发、安全
- **SQL**：查询相关缺陷
- **Shell**：脚本相关问题

### 报告生成

该技能生成全面的 Markdown 报告，包括：

1. **执行摘要**
   - 审查的文件/行数总数
   - 按严重程度分类的缺陷计数
   - 缺陷密度指标

2. **CWD 类别统计**
   - 按类别细分
   - 严重程度分布

3. **详细发现**
   - 文件路径和行号
   - 代码片段
   - 描述和修复建议

4. **建议**
   - 优先修复列表
   - 基于类别的分组

## 使用方法

### 基本审查

```
请审查 src/ 中的代码是否存在 CWD 缺陷。
```

### 重点审查

```
审查 src/memory.c 中与内存相关的缺陷（CWD-1002, CWD-1003）。
```

### 安全扫描

```
使用 CWD 模式扫描代码库中的安全漏洞。
```

### 完整分析并生成报告

```
执行完整的 CWD 代码审查并生成详细报告。
```

## 集成

本技能可与其他编码指南配合使用：

```markdown
配合使用：java-guide, c-guide, cpp-guide, python-guide, js-ts-guide
```

## 性能说明

- 引用根据检测到的语言延迟加载
- 模式匹配针对常见缺陷模式进行了优化
- 大型代码库分块处理以获得更好性能

## 局限性

1. 基于模式的检测可能产生误报
2. 不能替代全面的安全测试
3. 某些缺陷需要运行时分析
4. 报告准确性取决于代码清晰度
